删除pip安装缓存

Posted on 2019-07-02 Edited on 2023-05-30 In Python , pip

删除以下路径文件

Linux and Unix

~/.cache/pip # and it respects the XDG_CACHE_HOME directory.

OS X

~/Library/Caches/pip

Windows

%LocalAppData%\pip\Cache
